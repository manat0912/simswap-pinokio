# SimSwap Pinokio Launcher

This is a 1-click launcher for SimSwap, an efficient framework for high fidelity face swapping.

## Features

- Image Face Swapping (224 and 512 resolutions)
- Video Face Swapping (224 and 512 resolutions)
- Gradio Web Interface
- Automatic model downloads
- Cross-platform support (Windows, Linux, Mac)

## Installation

Click the **Install** button in Pinokio.

## Usage

1. Click **Start** to launch the Gradio web interface.
2. Once the interface is loaded, click **Open Web UI**.
3. Upload a **Source Image** (the face you want to use).
4. Upload a **Target Image** or **Target Video**.
5. Choose the **Crop Size** (224 for standard, 512 for HQ).
6. Click **Swap** and wait for the result.
